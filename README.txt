We have added the following functionality:
1. Around the board pacman can find heart shaped "medicines". If they will be taken by him, he will
get more life for each one.
2.We have added animated gifs when the user loses or wins.

Please notice that when choosing the keys in the settings page, you must first press on the key we wish to set and than another keyboard key to your choice. 
This key will now be associated to the movement you have just set.
